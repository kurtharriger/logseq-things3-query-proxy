# shadow-electron-starter
ClojureScript + Shadow-cljs + Electron + Reagent

## How to Run
```
npm install

npm run dev
npx electron .
```

## Release
```
npm run build
npx electron-packager . HelloWorld --platform=darwin --arch=x64
```
